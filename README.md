1️⃣ What is the difference between var, let, and const?
\*\*\* var is old and has no block scope (so it can be a problem), let is the new and improved var that respects blocks, and const is like let but once you set it you can't change it;

2️⃣ What is the spread operator (...)?
\*\*\* The spread operator (...) is like taking items out of an array or object and spreading them out like flour;

3️⃣ What is the difference between map(), filter(), and forEach()?
**_ map() creates a new array by transforming each item, filter() creates a new array with only items that pass a test, and forEach() just loops through items without creating a new array;
4️⃣ What is an arrow function?
_** An arrow function () => {} is a shorter way to write functions that also behaves differently with "this";
5️⃣ What are template literals?
\*\*\* Template literals are strings with backticks `` that let you put variables directly inside with ${} and write in multiple lines without using + signs.
